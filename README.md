# hupubao-common

简单封装的公共模块，方便开发使用

**包含：**

- 日期工具
- 字符串工具
- 驼峰转换工具
- 断言工具
- 数组工具
- list工具
- 二维码工具
- redis工具
- 反射工具
- 资源目录文件读取工具
- 金额：元，默认精度为2
- http访问工具



**其中包含的依赖有:**
